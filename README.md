# python_aasignment_1
This repo contain completion of my first assignment of the python course. there are two task included in this assignment as follow.


**Task 1**

In this task:
- Program takes two numbers as input from the user.
- It performs following simple arithmatic operations:
     - Addition
     - Subtraction
     - Multiplication
     - Division
- It displays result of all basic arithmatic operations of two number.

Program for this task is available in file 'mathop.py'.

**Task 2**

In this task:
- Program takes first name and last name of the user as input.
- It concatenates first name and last name into full name.
-  It displays a greeting message to the user:
  
  "hello 'full name'! Welcome to the python program."

 Program for this task is available in file 'greet.py'.
